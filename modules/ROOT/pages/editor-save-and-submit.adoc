== Adding save functionality to the editor

\{\{site.productname}} can be configured to allow users to save the editor content. For information on configuring the user saving, see: link:{baseurl}/plugins-ref/opensource/save/[The Save plugin].

== Autosaving the editor content

\{\{site.productname}} can be configured to automatically save the editor content. For information on configuring the automatic saving, see: link:{baseurl}/plugins-ref/opensource/autosave/[The Autosave plugin].
