= Creating custom Group toolbar buttons

:title_nav: Group toolbar button

:description: Creating custom Group toolbar buttons for TinyMCE
:keywords: toolbar toolbarbuttons buttons toolbarbuttonsapi

A group toolbar button is a toolbar button that contains a collection of other toolbar buttons that are displayed in a pop-up when clicked. The style of toolbar shown is based on the current link:{baseurl}/interface/toolbars/toolbar-configuration-options/#toolbarmode[toolbar mode]. For example, if `+toolbar_mode+` is set to `+floating+`, the toolbar group pop-up will appear in a floating shelf.

____
*Note:* The group toolbar button is _only_ supported when using the `+floating+` toolbar mode. If the `+toolbar_groups+` option is used with other toolbar modes, the toolbar group button will not be displayed and a warning message will be printed in the console.
____
