
| Command         | Description                        |
| --------------- | ---------------------------------- |
| mceDirectionLTR | Changes the directionality to LTR. |
| mceDirectionRTL | Changes the directionality to RTL. |

**Examples**

```js
tinymce.activeEditor.execCommand('mceDirectionLTR');
tinymce.activeEditor.execCommand('mceDirectionRTL');
```
