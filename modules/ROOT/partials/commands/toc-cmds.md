
| Command      | Description                                            |
| ------------ | ------------------------------------------------------ |
| mceInsertToc | Inserts a Table of Contents into the editor.           |
| mceUpdateToc | Updates an existing Table of Contents.                 |

**Examples**

```js
tinymce.activeEditor.execCommand('mceInsertToc');
tinymce.activeEditor.execCommand('mceUpdateToc');
```
